[필수] 기본 회원정보 입력

이메일 인증을 포함한 모든 항목을 입력하셔야 가입이 완료됩니다.

  • 이메일
  • @

    로그인 아이디 및 전형 관련 안내메일을 보내는데 사용됩니다. 반드시 실제로 사용하는 이메일 주소를 입력해주세요.

  • 이메일 확인
  • 인증번호 받기 확인

    인증번호 받기를 클릭하시면, 입력하신 이메일 및 보조이메일로 인증메일이 전송됩니다.
    이메일로 발송되는 인증코드를 5분 이내에 입력하여 주세요.

  • 보조 이메일
  • @

    이메일 주소 분실 시 사용자 인증을 위한 보조 이메일 주소입니다.
    hotmail, Outlook 사용자는 메일 수신이 어려울 수 있사오니, 가급적 다른 계정을 알려주세요.

  • 한글이름

  • 한글만 입력가능합니다.

  • 영문이름

  • 영문만 입력가능합니다.

  • 생년월일
  • 비밀번호

  • 특수문자(~!#$%^*) 및 영문과 숫자를 포함하여 8~12자로 설정해주세요.

  • 비밀번호 확인
  • 휴대폰 번호
[선택] 관심 분야 설정

관심있는 회사, 업무, 근무지, 전공을 등록하면 관련 공고를 자동으로 추천하여 알려드립니다.

(최대 3개까지 등록 가능, 설정일로부터 6개월까지 알림 정보 제공)

  • 관심회사
  • 관심업무
  • 관심근무지
  • 관련전공

  • 선택한 관심정보에 따라 자동 추천되는 공고를 휴대폰이나 이메일로 알림 받으시겠어요?
